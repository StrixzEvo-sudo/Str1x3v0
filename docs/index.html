Windows Temp Cleaner

Clean temp files without turning it into a whole install process.

System Cleanup Utility clears the usual Windows temp folders, goes further with optional cache cleanup, handles admin-only locations, and can update itself from GitHub Releases.

Distribution Single-file `.exe`

Safe by default

It clears folder contents while keeping the folders themselves, and it skips symlinks and reparse points.

Advanced cache targets

It can optionally clean browser cache, shader cache, Store app temp cache, thumbnail DBs, and Windows error reports.

Updates without a launcher

The `.exe` checks GitHub Releases at startup and can swap itself out when a newer build exists.

Use It

Three steps on the user side.

  1. Download the latest `.exe` from this page.
  2. Run it as administrator when prompted.
  3. Pick standard temp targets or enable advanced cache cleanup, then start the cleanup.

Close browsers and launchers before running advanced cache cleanup. Prefetch and other caches are rebuilt as Windows and apps need them.

Hosted on GitHub Pages. Releases are served through GitHub Releases.
